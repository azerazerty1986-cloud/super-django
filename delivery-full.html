# الشركة المسؤول الهاتف النوع الولاية الإجراء

دليل الاستخدام السريع

هذا النظام يدعم جلب البيانات من ملفات JSON و CSV لضمان أقصى درجات الدقة والسرعة.

صيغة JSON المطلوبة: 
[
  {
    "company": "اسم الشركة",
    "manager": "المسؤول",
    "phone": "0555123456",
    "type": "delivery",
    "state": "الجزائر"
  }
]
صيغة CSV المطلوبة: 
company,manager,phone,type,state
شركة البراق,أحمد,0555123456,delivery,الجزائر

💡 نصيحة: ثبت ملف JSON أو CSV في قناتك على تليجرام، ثم اضغط على زر "جلب من المثبتة" لتحديث البيانات فوراً.
